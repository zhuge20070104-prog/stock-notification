# 自动化交易决策系统开发计划 (PLAN.md)

本方案基于视频图片中的五大核心市场场景（正常回调、恐惧回调、极端恐慌、系统性风险、过度贪婪），构建一个自动获取宏观及情绪指标、并实时输出仓位买卖建议（30%/30%/40%等比例分配）的自动化逻辑框架。本版本已引入 SPY 与 RSP 的背离分析，用以量化大盘集中度风险。

---

## 核心指标获取管道 (Data Pipeline)

为了让 Claude Code 或自动化脚本能做出判断，必须先通过 API 或库稳定获取以下基础数据：

| 指标分类 | 核心指标名称 | 推荐获取渠道 / 接口 / Python库 | 作用说明 |
| :--- | :--- | :--- | :--- |
| **波动率** | **VIX 指数** | `yfinance` 库（代码：`^VIX`） | 衡量市场恐慌程度与筑底/筑顶信号 |
| **情绪面** | **恐惧与贪婪指数** | 爬取 CNN Fear & Greed 网页或第三方开源 API | 识别市场情绪的极端冰点与沸点 |
| **散户情绪**| **AAII 散户看跌比例** | 通过 FRED 数据集或定投网站接口获取 | 捕捉散户极端悲观的逆向买入机会 |
| **技术面** | **主要指数回调比例** | `yfinance` 获取 `^GSPC` (标普500) | 计算 `(当前价 - 252日最高点) / 252日最高点` |
| **集中度** | **大盘广度 / 集中度** | `yfinance` 获取 `SPY` 与 `RSP` | **核心指标**：对比市值加权与等权重加权，量化缩量抱团风险 |
| **信用债** | **HYG/JNK 价格与利差**| `yfinance` 获取 `HYG`；FRED 获取 `BAMLH0A0HYM2` | 监控信用市场是否暴雷（识别真摔与系统性风险） |
| **宏观面** | **美元指数 (DXY)** | `yfinance` 获取 `DX-Y.NYB` | 观察全球流动性是否出现失控式抽水 |

## 五大场景量化决策引擎 (Decision Engine)

### 场景一：正常回调（可以正常定投）
* **触发条件**：
    * `18 <= VIX <= 25` （波动升温，开始留意）
    * `25 <= 恐惧贪婪指数 <= 45` （进入恐惧区间）
    * `3% <= 指数从高点回调比例 <= 5%`
    * `信用市场`（HYG/利差）表现稳定，跨资产无异常
* **动作逻辑**：**保持正常定投频率**，不触发额外的大额加仓。

### 场景二：恐惧回调（分批加仓机制）
* **触发条件**：
    * `25 < VIX <= 30` （恐慌升温，接近30）
    * `恐惧贪婪指数 < 25` （进入极度恐惧区间）
    * `7% <= 指数从高点回调比例 <= 10%`
    * `系统安全确认`：HYG/JNK 趋势未崩溃，且美元指数（DXY）没有失控式暴涨
* **资金分配算法 (30% / 30% / 40%)**：
    1. **第一笔 (30%)**：当系统判定进入该场景，且满足 `VIX > 25` 时，即刻买入。
    2. **第二笔 (30%)**：若大盘继续下杀，`VIX` 冲到 `30` 附近时，自动补入第二笔。
    3. **第三笔 (40%)**：等待 `VIX` 从高位开始回落，且市场广度指标（如 RSP 止跌转涨）开始改善时，补入最后 40% 资金。

### 场景三：极端恐慌（强烈关注逆向抄底）
* **触发条件**：
    * `VIX > 35 或 VIX > 40` （市场进入极端恐慌）
    * `0 <= 恐惧贪婪指数 <= 15` （恐慌指标处于绝对冰点）
    * `AAII 散户看跌比例 > 31%` （远高于历史均值，散户割肉完毕）
    * `核心底线`：信用市场未全面冻结，银行体系运行正常，无大型金融机构暴雷
* **动作逻辑**：**逆向大额买入，分段建仓**。
    * *风控核心*：分好买入区间，**坚决不一口气 All-in**。
    * *标的选择*：优先买入核心资产、现金流强劲的头部公司，必须保留一部分现金以防万一。

### 场景四：系统性风险（先防守、绝不抄底）
* **触发条件**：
    * `VIX` 持续狂飙居高不下，且完全没有见顶回落迹象
    * `银行股暴跌`，且美联储信用利差（Spreads）快速扩大（代表借贷市场冻结）
    * `HYG/JNK` 垃圾债价格破位断崖式大跌
    * `美元指数 (DXY)` 出现避险引发的暴力狂飙，美债市场流动性异常
* **动作逻辑**：**绝对防守，空仓或持有现金观望**。
    * 禁止任何加仓或定投行为！降低杠杆，减少高 Beta 资产，保留高额现金。静待信用市场企稳再考虑恢复投资。

### 场景五：过度贪婪（考虑减仓风险）
* **触发条件**：
    * `VIX < 15` 长期低迷（市场极度自满，防备黑天鹅）
    * `恐惧贪婪指数 > 75` （极度贪婪），散户情绪极度乐观，新手开始疯狂加杠杆
    * **【核心风控项】大盘集中度恶化**：`SPY` 指数创历史新高，但 `SPY - RSP` 的收益率差值持续扩大（如 20 日动量背离超过 3%）。即“只有少数几家巨头在拉大盘，绝大多数股票在跌”。
* **动作逻辑**：**降低进攻性，执行分批减仓策略**。
    * 卖出、阶段性结利涨幅过大的高估值个股，资金转向稳健的核心防御资产。
    * 可按 **30% 或 40%** 的比例阶梯式减仓，或通过卖出 `Covered Call` (备兑看涨期权) 锁定高位收益，提高现金比例。
